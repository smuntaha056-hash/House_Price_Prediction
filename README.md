#  House Price Prediction

##  Objective
This project aims to predict house prices using machine learning 
based on features like number of rooms and other housing factors.



##  Tools Used
- Python
- pandas, numpy
- matplotlib, seaborn
- scikit-learn

---

##  Project Steps
1. Data Loading and Exploration  
2. Data Visualization  
3. Feature Selection  
4. Model Training (Linear Regression)  
5. Model Evaluation  



##  Results
- The model achieved an R² score of ~0.7–0.75  
- Model predicts house prices with reasonable accuracy  

---

##  Key Insights
- Number of rooms has strong positive impact on price  
- Some features have weaker influence
